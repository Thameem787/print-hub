# 📦 PrintHub - The Future of Local Manufacturing

PrintHub is a high-end marketplace platform designed to bridge the gap between local 3D printing enthusiasts and customers. It enables a decentralized manufacturing network, allowing hardware owners to monetize their equipment while providing customers with fast, affordable, and locally-sourced 3D printing services.

## ✨ Key Features

- **Decentralized Manufacturing:** Connect directly with local printing professionals.
- **Hardware Monetization:** Turn your 3D printer into a revenue-generating asset.
- **Dynamic Portals:** Tailored experiences for both **Vendors** (to manage jobs and hardware) and **Customers** (to track orders and discover local talent).
- **Cinematic UI:** A premium, modern interface built with Framer Motion for smooth, interactive experiences.

## 🛠️ Tech Stack

- **Frontend:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** Modern CSS with a focus on Glassmorphism and Cinematic aesthetics.
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router 7](https://reactrouter.com/)

## 🚀 Getting Started

Follow these steps to get the project running locally on your machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (version 18 or higher recommended).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rizirfan/printhub.git
   cd printhub
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

### Running Locally

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the next available port).

## 📂 Project Structure

- `src/pages/LandingPage.tsx`: The high-impact entry point.
- `src/pages/VendorDashboard.tsx`: Management tools for printing partners.
- `src/pages/CustomerDashboard.tsx`: Order tracking and discovery for users.
- `src/App.tsx`: Main application logic and routing.
